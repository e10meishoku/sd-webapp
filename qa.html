Streamlit だけでも「触れる MVP」は十分作れる。 【構成】 ユーザー → Streamlit(Python) → 同一サーバー上の GPU 
import torch, diffusers as d, streamlit as st

pipe = d.StableDiffusionPipeline.from_pretrained(
    "YourModel", torch_dtype=torch.float16).to("cuda")

prompt = st.text_input("プロンプト")
if st.button("生成"):
    img = pipe(prompt, num_inference_steps=30).images[0]
    st.image(img)
